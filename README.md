# Ginger

A minimalist, organised way to track your to-do lists — built as a native macOS app.

Ginger keeps your tasks structured without clutter. Instead of one endless list, tasks live inside a clear hierarchy you can drill into, so each screen stays calm and focused.

## Why I'm building it

I wanted a to-do app that was minimal and organised. Ginger is my take: a clean nested structure for the different areas of life, with everything stored locally on your own machine.

## Tech stack

- **Electron** — desktop app shell
- **React** + **Vite** — UI
- **SQLite** — local data storage
- **Lucide** — icons
